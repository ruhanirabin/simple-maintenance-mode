1 Simple Maintenance Mode plugin allows you to enable or disable maintenance mode while you are working on your WordPress site.

== Description ==

This plugin is designed to help you put your WordPress site into maintenance mode while you are working on it. It allows you to enable or disable maintenance mode with ease, and comes with a number of useful features to help you manage your site while it is under construction.

### With 1 Simple Maintenance Mode plugin, you can:

- Enable or disable maintenance mode with a single click
- Show a persistent admin notice while maintenance mode is on
- Exclude some pages for open access if you like to show a specific page to the public view
- Auto updates via GitHub releases

== Installation ==

### To install 1 Simple Maintenance Mode plugin, follow these steps:

Upload the plugin files to the /wp-content/plugins/1-simple-maintenance-mode directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' screen in WordPress
Use the Settings->1 Simple Maintenance Mode screen to configure the plugin
== Frequently Asked Questions ==

= How do I enable maintenance mode? =

To enable maintenance mode, simply click the "Enable Maintenance Mode" button on the 1 Simple Maintenance Mode settings page. This will put your site into maintenance mode, and display a notice to your visitors that your site is currently under construction.

= How do I exclude pages from maintenance mode? =

To exclude pages from maintenance mode, simply enter the page IDs or slugs in the "Exclude Pages" field on the 1 Simple Maintenance Mode settings page. This will allow visitors to access those pages even when your site is in maintenance mode.

= How do I disable maintenance mode? =

To disable maintenance mode, simply click the "Disable Maintenance Mode" button on the 1 Simple Maintenance Mode settings page. This will take your site out of maintenance mode, and allow visitors to access your site as normal.
